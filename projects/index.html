Second hand

개요

당근마켓을 모티브로 만든 중고거래 플랫폼입니다. oAuth 로그인, 아이템 게시,

  • 기간 : 23.06 ~ 23.08
  • 팀원 : Backend 2명, Frontend 2명, iOS 2명
  • 주요 기술 : Java, Springboot, Spring data JPA, MySQL, Redis
  • repository : link
  • api docs : link
  • web : http://fivehands-but-iiiiii.site/

주요 구현사항 및 이슈

  • DB 최적화를 위해 테이블 모델링시 부분 반정규화를 진행하였습니다.
  • Mysql에서 인덱스 설정과 실행계획 분석 후 쿼리 튜닝하였습니다.
  • github action과 docker compose를 이용하여 빌드 배포 과정을 자동화하였습니다.
  • Nginx의 Reversed-Proxy를 이용하여 로드밸런싱하였습니다.
  • STOMP, Redis pub/sub 을 이용해 실시간 채팅 시스템을 구현하였습니다.
  • 채팅 알람을 SSE 이용하여 구현하였습니다.
  • AWS Lambda를 이용하여 이미지 리사이징 작업을 서버 외부에서 처리하여 서버 부하를 줄이고자 하였습니다.
  • Spring Event를 이용하여 어플리케이션 내 이벤트를 비동기로 처리하여 도메인간 의존성을 낮추었습니다.
